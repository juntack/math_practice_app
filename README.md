# けいさん れんしゅう

小学1年生向けの、足し算・引き算の練習用Webアプリです。

公開URL: [https://juntack.github.io/math_practice_app/](https://juntack.github.io/math_practice_app/)

## 使い方

`index.html` をブラウザで開いてください。インストールは不要です。

- 足し算は答えが10まで、引き算は10から引く問題まで出題します。
- 同じ練習中は問題が重複しません。候補をすべて出題した場合は、次の問題からランダムな順で繰り返します。
- 足し算のみ、引き算のみ、両方からの出題を選べます。
- 問題数（1〜100問）と、時間を計測するかどうかを設定してからスタートできます。計測時間は終了時だけ表示されます。
- 画面内の数字ボタンで答えを入力して「はんていする」を押すと、正解なら `○`、不正解なら `×` を表示します。iPadの標準キーボードは開きません。
- 正解時はチャイム音、不正解時はブザー音が鳴ります。
- `N` キーは、回答前は判定、判定後は次の問題へ進む操作として使えます。判定後は Enter キーでも次の問題へ進めます。
- 「せいせきを リセット」で正解数と回答済み問題数を0に戻せます。
